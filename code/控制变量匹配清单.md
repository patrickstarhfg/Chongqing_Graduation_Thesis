## 2. PDF 中提到的特殊指标 (Potential Missing)

PDF 中提到了两类可能需要补充的指标：

### (1) 市场环境与区域经济指标
*   **原文**：*“4. 市场环境与区域经济指标”*
*   **现状**：我们目前只有 **GDP**。
*   **建议补充**：如果想更全面，可以考虑加入：
    *   **市场化指数 (Marketization Index)**：樊纲指数（通常需单独下载）。
    *   **赫芬达尔指数 (HHI)**：衡量行业竞争程度（可以用 CSMAR 的行业收入数据自己算）。

### (2) 数据要素相关指标
*   **原文**：*“2. 数据要素相关指标...文本分析词典”*
*   **现状**：我们目前使用了 `DigitalScore`（数字化转型程度）。
*   **匹配度**：✅ **完全匹配**。我们的 `DigitalScore` 正是基于年报 MD&A 文本分析得出的，涵盖了“数据采集”、“数据处理”、“数据应用”等关键词，完全符合 PDF 的要求。
