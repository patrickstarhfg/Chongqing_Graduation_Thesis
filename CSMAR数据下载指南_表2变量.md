# CSMAR 数据库详细下载指南 (基于表2变量)
——针对《基于因果森林的数据要素赋能对企业全要素生产率的异质性影响研究》

根据您提供的变量定义表（表2），我为您整理了在 CSMAR 数据库中下载对应数据的详细步骤。请登录账号后按以下模块操作。

## 第一部分：财务与市场指标 (Financials & Market)
**涉及变量**：`ROA`, `Size`, `Lev`, `Grow`, `TobinQ`, `Age`

1.  **路径**：数据中心 -> **股票市场系列** -> **股票市场交易** (或者直接搜 **"相对价值指标"**)
    *   **表名**：**相对价值指标 (Relative Value)**
    *   **勾选字段**：
        *   `TobinQ`：托宾Q值 (TobinQ) —— *通常建议选择 B (流通股市值+非流通股股份数×每股净资产+负债账面价值)/总资产 或 C*
2.  **路径**：数据中心 -> **公司研究系列** -> **财务指标分析**
    *   **表名**：**财务指标分析 (Financial Ratios)**
    *   **勾选字段**：
        *   `ROA`：总资产净利润率 (ROA) —— *对应定义：净利润/年末总资产*
        *   `Lev`：资产负债率 (DebtsAssetRatio) —— *对应定义：年末总负债/总资产*
        *   `Grow`：营业收入增长率 (OpeIncGroRate) —— *对应定义：(当年营收-上年营收)/上年营收*
3.  **路径**：数据中心 -> **公司研究系列** -> **财务报表** -> **资产负债表**
    *   **表名**：**资产负债表 (Balance Sheet)**
    *   **勾选字段**：
        *   `Size`：资产总计 (TotalAssets) —— *下载后需取自然对数 (ln)*
4.  **路径**：数据中心 -> **股票市场系列** -> **股票基本情况**
    *   **表名**：**股票基本情况表 (Stock Profile)**
    *   **勾选字段**：
        *   `Age`：上市日期 (ListingDate) —— *下载后需计算：ln(当年年份 - 上市年份)*

## 第二部分：公司治理指标 (Corporate Governance)
**涉及变量**：`Duality`, `Board`, `Indb`, `Top1`, `SOE`

1.  **路径**：数据中心 -> **公司研究系列** -> **公司治理**
2.  **子库与表名**：
    *   **表1：董事会特征 (Board Characteristics)**
        *   `Board`：董事人数 (BoardSize) —— *下载后需取自然对数 (ln)*
        *   `Indb`：独立董事人数 (IndBoard) —— *需计算占比：独立董事人数/董事人数*
        *   `Duality`：董事长与总经理兼任情况 (IsDual) —— *通常 1=兼任, 0=不兼任*
    *   **表2：十大股东文件 (Top 10 Shareholders)**
        *   `Top1`：第一大股东持股比例 (HoldRatio1)
    *   **表3：实际控制人文件 (Actual Controller)** (或者在 **"股权性质"** 文件中)
        *   `SOE`：实际控制人性质 (ControllerNature)
            *   *注：需根据代码转换，通常 1100/1000=国资，2000/2100=民营等。若为国企赋值1，否则0*

---

## 第三部分：数字化转型 (Digital Transformation)
**涉及变量**：`Treat_time`

1.  **路径**：数据中心 -> **数字经济系列** -> **上市公司数字化转型** (如果学校购买了此库)
    *   **表名**：**数字化转型程度**
    *   **勾选字段**：
        *   `Treat_time`：下载 **"数字化转型总词频"** 或 **"是否进行数字化转型"**。
        *   *处理方法：若 MD&A 出现数字化词典中词汇（词频>0），则取1，反之为0*
2.  **替代方案 (如果没买数字经济库)**：
    *   路径：**公司研究系列** -> **财务报表附注** -> **管理层讨论与分析 (MD&A)**
    *   下载 MD&A 文本内容，需使用 Python 结合“数字化词典”进行关键词匹配和计数。

---

## 第四部分：区域经济指标 (Regional Economy)
**涉及变量**：`GDP`

1.  **路径**：数据中心 -> **宏观经济系列** -> **区域经济**
    *   **表名**：**各地区主要经济指标 (Regional Economy)**
    *   **勾选字段**：
        *   `GDP`：地区生产总值 (GDP)
    *   **注意**：
        1.  这是省级或市级数据，下载后需要通过 **"注册地所在省份/城市"** 与上市公司数据进行匹配。
        2.  *下载后需取自然对数 (ln)*

---

## 📝 数据下载与合并贴士
1.  **主键**：所有表下载时务必勾选 **股票代码 (Stkcd)** 和 **会计年度 (Year)**，这是后续合并数据的依据。
2.  **频率**：均选择 **"年 (Annual)"** 数据。
3.  **时间范围**：建议选择 **2011-2023** (覆盖数字化转型热门时期，且保证数据较新)。
4.  **单位**：CSMAR 下载时注意金额单位（元/万元/百万元），合并前需统一换算（例如 GDP 通常单位较大，需注意与企业规模的单位协调）。
